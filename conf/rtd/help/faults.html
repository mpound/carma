CARMA Faults Window Help

This window displays information about the present faults in the system.

Fault names that are colored red are presently causing the alarm to sound (unless the alarm is presently globally disabled).

Please note that some root faults do not cause blanking or flagging. At last check these included:

Imr.numNotRunningCriticalServers
Control.Subarray1.scriptState
Control.Subarray1.alarm
OvroN.Drive.state
BimaN.Drive.state
So if you see one or more of these faults in this RTD window, you will not see associated blanking/flagging in the Fault System RTD window.
