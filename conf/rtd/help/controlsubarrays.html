Control Status Window Help

This window displays control information for all carma subarrays. Each folder represents subarray information for the named subarray. Shows subarray status (active/inactive), source, antenna status and correlator status.

Note that the reachable status will only return to green if:

  1. An operation involving the remote object is attempted.
  2. The monitor subsystem associated with the remote object is “alive”.
  3. The operation succeeds in being dispatched.

One way to artifically force an operation to be attempted on all presently disconnected remote objects is to call s.reconnect(False) from a sac session.
