This is an installation of the shave tool, which simplifies automake output. It
is being used in place of the built-in automake functionality present in
>=automake-1.11. Once CARMA has upgraded to this version of automake, this
>support code can be removed.

It was retrieved from git://git.lespiau.name/shave on 2009-08-27. These files
come from git revision 2134bb12.
