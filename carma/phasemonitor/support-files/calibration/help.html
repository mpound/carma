Calibration Plots Description

A known problem exists with the raw voltages output by the correlators for the phase monitor, for more information about this see: Known problems/features.

As the phase monitor software runs, it is recording the raw voltages returned by the two arms of the phase monitor correlators to disk. At noon each day, the last 86400 data points (24 hours of data) are copied to the calibration area, then re-run through the phase monitor software twice.

The first run, applies no correction to the voltages and produces the regular 1 second data normally produced by the copy of the regularly running phase monitor. This is then used to produce a plot of the complex voltages. This exposes the tilted ellipse with offset described in the Known problems above.

The second run, applies current correction parameters to the voltages and produces the regular 1 second data and a corrected plot is then produced.

This can help diagnose problems with the system. If an ellipse is smaller than it used to be, this means that the amplitudes are lower and action should be taken to correct this. If the corrected version is not more or less circular and centered on the origin, this means the current parameters are not properly correcting and need to be recomputed. As of 07 Dec 2007, there is a plan to automatically recompute these parameters once every 24 hours, but is not yet implemented.
